### Draggable Class Usage Example (JavaScript)

Source: https://leafletjs.com/reference-2.0.0

Demonstrates how to instantiate the Draggable class and enable dragging for a DOM element. This is the primary way to integrate the Draggable functionality into a web page.

```javascript
const elementToDrag = document.getElementById('yourElementId');
const draggable = new Draggable(elementToDrag);
draggable.enable();
```
