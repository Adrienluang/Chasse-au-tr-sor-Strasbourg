### Leaflet Fly-to Animation and Fractional Zoom (JavaScript)

Source: https://leafletjs.com/2016/09/27/leaflet-1.0-final

Illustrates the use of `map.flyTo` for animated map transitions and fractional zoom levels (`map.setZoom`). These enhance user experience by providing smooth navigation.

```javascript
// Fly to a specific location and zoom level
map.flyTo([51.505, -0.09], 13);

// Set a fractional zoom level
map.setZoom(12.34);
```
