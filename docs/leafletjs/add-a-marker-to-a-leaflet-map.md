### Add a Marker to a Leaflet Map

Source: https://leafletjs.com/examples/quick-start

This code snippet demonstrates how to add a basic marker to a Leaflet map. It requires a pre-initialized Leaflet map object. The function takes latitude and longitude coordinates and adds a marker at that location.

```javascript
var marker = L.marker([51.5, -0.09]).addTo(map);
```
