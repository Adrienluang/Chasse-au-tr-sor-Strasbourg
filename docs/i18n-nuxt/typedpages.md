### `typedPages`

- type: `boolean`{.shiki,shiki-themes,material-theme-lighter,material-theme,material-theme-palenight lang="ts-type"}
- default: `true`{.shiki,shiki-themes,material-theme-lighter,material-theme,material-theme-palenight lang="ts"}
- Generates route types used in composables and configuration, this feature is enabled by default when Nuxt's `experimental.typedPages` is enabled.

::callout{color="warning" icon="i-heroicons-exclamation-triangle"}
This feature relies on [Nuxt's `experimental.typedPages`](https://nuxt.com/docs/guide/going-further/experimental-features#typedpages){rel=""nofollow""} and will not work if this is not enabled.
::
