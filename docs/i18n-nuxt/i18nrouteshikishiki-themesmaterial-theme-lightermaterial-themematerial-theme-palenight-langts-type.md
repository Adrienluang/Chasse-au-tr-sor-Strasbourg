### `I18nRoute`{.shiki,shiki-themes,material-theme-lighter,material-theme,material-theme-palenight lang="ts-type"}

An object accepting the following i18n route settings:

- **`paths`**
  - **Type**: `Record<Locale, `/${string}`>`:br Customize page component routes per locale. You can specify static and dynamic paths for vue-router.
- **`locales`**
  - **Type**: `Locale[]`:br Some locales to which the page component should be localized.
