### `routeBaseName()`{.shiki,shiki-themes,material-theme-lighter,material-theme,material-theme-palenight lang="ts"}

- **Arguments**:
  - route (type: `string | Route`{.shiki,shiki-themes,material-theme-lighter,material-theme,material-theme-palenight lang="ts-type"})
- **Returns**: `string`{.shiki,shiki-themes,material-theme-lighter,material-theme,material-theme-palenight lang="ts-type"}

Returns base name of the passed route. The base name of a route is its name without a locale suffix or other metadata added by `@nuxtjs/i18n`.
